# 🏰 Dungeon Adventure Game (Text-Based)

## 🎮 About the Project

This is a **text-based dungeon adventure game** built using Python.
The player explores different dungeons, makes choices, and reaches different endings based on their decisions.

Each dungeon has:

* A unique story
* Multiple choices
* Different outcomes

---

## 🚀 Features

* 🏰 5 Unique Dungeons
* 🎯 Choice-based gameplay
* 📖 Story-driven experience
* ⚡ Multiple endings
* 🧠 Decision-making impact
* 😂 Funny moments combined

---

## 🗺️ Dungeons Included

1. **Abandoned Entrance**
   A mysterious place with ghostly whispers and hidden secrets.

2. **Dark Maze**
   A dangerous labyrinth filled with goblins, traps, and risky choices.

3. **Frozen Prison**
   A cold prison where your moral decisions decide your fate.

4. **Shadow King’s Domain**
   A psychological dungeon testing your will against darkness.

5. **Dragon’s Core**
   The "hardest" dungeon... or is it? 👀
